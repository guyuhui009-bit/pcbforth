# PCBforth 网站设计理念

## 参考来源
linkytech.com — PCB设计、SMT焊接、制板、元器件供应一站式服务商

---

## 三种设计方向

### 方案A: 工业精密风 (Industrial Precision)
深色背景配合蓝色荧光线路，类似PCB板的绿/蓝底色，强调技术感和精密制造。
概率: 0.07

### 方案B: 科技蓝图风 (Blueprint Tech) ← 选定
深海军蓝底色，白色PCB走线图案作为背景纹理，左侧固定导航栏，右侧内容区。
概率: 0.08

### 方案C: 现代工程风 (Modern Engineering)
白底+蓝色强调色，简洁现代，适合B2B商务场景。
概率: 0.04

---

## 选定方案：Blueprint Tech（蓝图科技风）

### Design Movement
工业蓝图美学 × 现代科技感 — 参考电路板蓝图、工程制图的视觉语言

### Core Principles
1. **PCB线路纹理背景** — 深蓝底色上的半透明电路走线图案，贯穿全站
2. **左右分栏布局** — 左侧固定导航菜单，右侧内容区域，类似工程软件界面
3. **精准的技术信息层级** — 数据、指标、能力参数清晰呈现
4. **动态交互反馈** — 菜单高亮、内容切换动画体现技术精密感

### Color Philosophy
- 主色: #0A1628 (深海军蓝) — 背景底色，稳重专业
- 强调色: #1E90FF (道奇蓝) — 主要交互元素、高亮
- 辅助色: #00D4FF (青蓝) — 线路纹理、装饰元素
- 文字色: #E8F4FD (淡蓝白) — 主要文字
- 卡片色: rgba(255,255,255,0.05) — 毛玻璃卡片

### Layout Paradigm
- 全屏左右分栏：左侧 280px 固定导航栏 + 右侧内容区
- 导航栏含品牌logo、菜单项、语言切换
- 右侧内容区按板块展示，支持平滑滚动

### Signature Elements
1. SVG PCB走线图案背景（动态流光效果）
2. 蓝色发光边框卡片
3. 数字计数器动画（展示客户数、年限等）

### Typography System
- 标题: Orbitron (英文技术感字体) + 思源黑体
- 正文: Source Sans Pro + 思源宋体
- 数字: Rajdhani (工程数字感)

### Brand Essence
PCBforth — 专业PCB全流程服务商，为电子工程师提供从原理图到成品的一站式解决方案

### Brand Voice
- 标题示例: "精密驱动创新，从原理图到成品"
- CTA示例: "立即咨询方案"
- 禁用: "欢迎来到我们的网站"

### Signature Brand Color
#1E90FF — 道奇蓝，科技感强，区别于竞品的红色系
